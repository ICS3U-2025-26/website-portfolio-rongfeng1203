Overview

Unit 3 covered the core programming concepts in Python: input/output, loops, conditionals, data structures, and custom functions. Each section below shows a code example I wrote and a short reflection on what I learned.

1 — Input, Output, and Variables

In this topic I learned how Python handles user input and how to display output in different ways. The input() function always returns a string, so I had to use int() to convert it to a number when doing math. I also practised three different output formatting styles: string concatenation with +, the .format() method, and f-strings — which became my favourite because they keep the code readable and are easy to debug.

# Input, Output, and Variables

name = input("Enter your name: ")

print("Hello, world!")
print("Hello, " + name + "!")
print("Ni hao", name, "!", sep="---")   # custom separator
print("I love you", name)               # default comma separator

# Control how a line ends (default is newline)
print("This is the first part.", end="")
print(" This is the second part.")

# Take numeric input and compute age
year = int(input("Enter your birth year: "))
age = 2025 - year

ageYoung = input("Enter your age: ")

# f-string formatting
print(f"{name} is {age} years old.")

# str.format() method
print("{} is {} years young.".format(name, ageYoung))

2 — For Loops, Math, and Random Numbers

This topic taught me how to use for loops with range() to repeat actions a set number of times. The three-argument form range(start, stop, step) was especially useful for counting backwards or skipping values. I also learned how to import and use the random module to generate unpredictable numbers — something that became essential when I built games later in the unit.

# For Loops, Math, and Random Numbers

# Number each line
for i in range(10):
    print(f"{i+1}. Hello")

# Count down from 10 to 1
for i in range(10, 0, -1):
    print(i, end="...")
print("Liftoff!")

# Print a growing staircase of stars
for i in range(6):
    print("*" * (1 + i))

# y = 4x - 6 for x from -5 to 5
for x in range(-5, 6):
    y = 4 * x - 6
    print(f"When x={x}, y={y}")

# ---- Random numbers ----
from random import randint

x = randint(1, 100)
print(f"A random number from 1 to 100 is {x}")

# Let the user choose the board size
size = int(input("Enter the size of the square board: "))
col = randint(1, size)
row = randint(1, size)
print(f"A random position on the board: ({col}, {row})")

3 — If-Statements and While Loops

Conditionals and loops are the backbone of interactive programs. I learned to build branching logic with if / elif / else chains and to keep a program running until a condition is met using while. A key insight was combining both: the while loop keeps the game going, and if-statements inside it give the player feedback each guess. I also learned to track a "chances" counter to add a win/lose state.

# If-Statements and While Loops

from random import randint

# ---- If-statement: grade calculator ----
grade = int(input("Enter a grade as a percentage: "))
if grade >= 80:
    print("A")
elif grade >= 70:
    print("B")
elif grade >= 60:
    print("C")
elif grade >= 50:
    print("D")
else:
    print("F")

# ---- While loop: number guessing game ----
guess = int(input("Guess a number from 1 to 100: "))
answer = randint(1, 100)
chances = 6

if guess == answer:
    print("Congratulations! You guessed it right on the first try!")
while guess != answer and chances > 0:
    if guess < answer:
        print(f"Too low! You have {chances} guess(es) left.")
    else:
        print(f"Too high! You have {chances} guess(es) left.")
    guess = int(input("Try again: "))
    chances -= 1

if guess == answer:
    print(f"Correct! The number was {answer}.")
else:
    print(f"Out of guesses. The number was {answer}.")

4 — Strings and Lists

Strings and lists are Python's two most-used data containers. For strings I practised slicing, chaining methods like .replace() and .upper(), and searching with the in keyword. For lists I worked through all the common operations: append, insert, sort, pop, remove, and list comprehensions. One of the most useful tricks was "-".join(list) to turn a list back into a formatted string.

# Strings and Lists

# ---- String operations ----
s = "Spongebob Squarepants"
print(s.replace("a", "1").replace("pi", "po").upper())  # chained methods
new_s = s[:6] + (s[6] + s[13:15]) * 2 + s[-4] + s[9:]  # slicing
print(new_s)

# ---- List operations ----
l = ["A", "X", "B", "X", "C", "X"]
print("original:", l)

l.append("D")           # add to end
l.insert(2, "Y")        # insert at index 2
l.sort()                # sort alphabetically
l.pop(2)                # remove item at index 2
l.remove("X")           # remove first "X"
l[3] = "Z"              # change item at index 3
del l[0]                # delete item at index 0

joined = "-".join(l)    # list -> string with hyphen separator
print("joined:", joined)

# List comprehension: uppercase letters A-Z
alphabet = [chr(i) for i in range(65, 91)]
print("alphabet:", alphabet)

# Convert character to ASCII and back
n = ord('A')
c = chr(66)
print(f"ord('A') = {n},  chr(66) = '{c}'")

5 — Custom Functions

Custom functions let you wrap reusable logic into a named block and call it whenever needed. I learned the difference between functions that only print (no return value), functions that take parameters, and functions that compute and return a result. The most interesting example was roll_die(), which returns a list of two random numbers — showing that a function can return any data type. Defining a menu with multiple mini-game functions showed me how larger programs are organized into modular pieces.

# Custom Functions

import random

# Function with parameter, no return value
def greet_user(name):
    print(f"Hello, {name}! Nice to meet you.")

greet_user("Rongfeng")
greet_user("Valerie")

# Function with parameters + return value
def calculate_area(length, width):
    return length * width

length = int(input("Length: "))
width  = int(input("Width: "))
print(f"Area = {calculate_area(length, width)}")

# Returns a boolean
def check_age(age):
    return age >= 16

age = int(input("Enter your age: "))
print("You can drive." if check_age(age) else "You cannot drive.")

# No parameter, returns a list
def roll_die():
    a = random.randint(1, 6)
    b = random.randint(1, 6)
    return [a, b]

result = roll_die()
print(f"Dice roll: {result[0]} and {result[1]}")
print(f"Sum: {sum(result)}")

# Mini higher-lower game
def higher_lower():
    number = random.randint(1, 100)
    guess = None
    while guess != number:
        guess = int(input("Guess a number between 1 and 100: "))
        if guess < number:
            print("Higher!")
        elif guess > number:
            print("Lower!")
        else:
            print("Correct!")

higher_lower()
