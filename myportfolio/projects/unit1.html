Overview

I built a simple quiz using HTML, CSS, and JavaScript. The goal was to practice DOM manipulation, event handling, and basic state management.

Design Choices

  • Clear layout with one question at a time.
  • Accessible buttons and keyboard navigation.
  • Score displayed at the end with an option to retry.

What I Learned

  • How to listen for click events and update the DOM.
  • Structuring data for quiz questions and answers.
  • Basic styling and responsive images.
